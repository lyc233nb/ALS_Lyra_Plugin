# ALS_Lyra_Plugin

## How To Use

1. You need to download lyra starter game project from fab(or Epic launcher), this plugin depends on lyra starter game to work properly.
2. Drag and drop the folder ALS_Lyra_Plugin-main(download zip file from the repository, this is the root folder) to the location in lyrastartergame project: Plugins/GameFeatures/
3. Right click on the lyrastartergame.uproject, click generate visual studio files.
4. You need to have an ide-- visual studio or rider to compile the whole project.
